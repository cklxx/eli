# Bitable 背景知识与限制

## 资源层级关系

```
App (多维表格应用)
 ├── Table (数据表) ×100
 │    ├── Record (记录/行) ×20,000
 │    ├── Field (字段/列) ×300
 │    └── View (视图) ×200
 └── Dashboard (仪表盘)
```

## 筛选条件 operator 列表

| operator | 含义 | 支持字段 | value 要求 |
|----------|------|----------|-----------|
| `is` | 等于 | 所有 | 单个值 |
| `isNot` | 不等于 | 除日期外 | 单个值 |
| `contains` | 包含 | 除日期外 | 可多个值 |
| `doesNotContain` | 不包含 | 除日期外 | 可多个值 |
| `isEmpty` | 为空 | 所有 | 必须为 `[]` |
| `isNotEmpty` | 不为空 | 所有 | 必须为 `[]` |
| `isGreater` | 大于 | 数字、日期 | 单个值 |
| `isGreaterEqual` | 大于等于 | 数字（不支持日期） | 单个值 |
| `isLess` | 小于 | 数字、日期 | 单个值 |
| `isLessEqual` | 小于等于 | 数字（不支持日期） | 单个值 |

**日期字段特殊值**: `["Today"]`, `["Tomorrow"]`, `["ExactDate", "时间戳"]`

## 使用限制

| 限制项 | 上限 |
|--------|------|
| 数据表 + 仪表盘 | 100（单个 App） |
| 记录数 | 20,000（单个数据表） |
| 字段数 | 300（单个数据表） |
| 视图数 | 200（单个数据表） |
| 批量创建/更新/删除 | 500（单次 API 调用） |
| 单元格文本 | 10 万字符 |
| 单选/多选选项 | 20,000（单个字段） |
| 单元格附件 | 100 |
| 单元格人员 | 1,000 |

## 其他约束

- 从其他数据源同步的数据表，**不支持增删改**记录
- 公式字段、查看引用字段是**只读**的
- 删除操作**无法恢复**
- 视图筛选条件使用 `field_id`，需先调用 field.list 获取

## 字段 Property 配置详解

创建/更新字段时需要的 `property` 参数结构因字段类型不同而异。收到 `125408X` 错误码时查阅此节。

常见字段 property：
- **单选/多选**：`property: { options: [{ name: "选项1" }, { name: "选项2" }] }`
- **进度**：`property: { min: 0, max: 100 }`
- **关联**：`property: { table_id: "tbl..." }`
- **货币**：`property: { currency_code: "CNY" }`

## 记录值数据结构详解

收到 `125406X` 错误码时查阅此节。

| type | 字段类型 | 值格式 | 说明 |
|------|----------|--------|------|
| 1 | 文本 | `"字符串"` | 纯文本 |
| 2 | 数字 | `123.45` | 数值类型 |
| 3 | 单选 | `"选项名"` | 字符串，非数组 |
| 4 | 多选 | `["选项1", "选项2"]` | 字符串数组 |
| 5 | 日期 | `1674206443000` | 毫秒时间戳 |
| 7 | 复选框 | `true` / `false` | 布尔值 |
| 11 | 人员 | `[{id: "ou_xxx"}]` | 数组对象，只传 id |
| 15 | 超链接 | `{link: "...", text: "..."}` | 对象格式 |
| 17 | 附件 | `[{file_token: "..."}]` | 需先上传获取 token |
| 18 | 单向关联 | `[{record_id: "rec..."}]` | 关联记录 ID |
| 22 | 地理位置 | `{location: "...", full_address: "..."}` | 位置对象 |
